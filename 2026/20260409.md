# Snapdragon + 16GiB RAMでローカルAIにWeb検索を実装した（LM Studio + MCP）

Gemma 4 が登場して、ローカル LLM がひさしぶり熱くなった感がありますが、

自分としては、

* ARM64（Snapdragon）で動くのか？
* 軽量モデルで実用になるのか？
* Web検索などのツール連携はできるのか？

という点が気になっていました（不満でした）。

結論から言うと、

> Snapdragon（ARM64）・RAM 16GiB の Windows 11 マシンで
> Gemma 4 E4B + MCP（Web検索）によるローカルAIチャットが実用的に動作しました。

本記事では、その構築手順をまとめます。

全体としては、 LM Studio で Gemma 4 を実行する際に、別途インストールした web-serach-mcp サーバを呼び出す、という仕組みになっています。

```
LM Studio（LLM）
   ↓
MCPクライアント（LM Studio内蔵）
   ↓
Web Search MCP サーバ
```

※ MCP（Model Context Protocol）は、LLMから外部ツールを呼び出すための仕組みです。

---

## 環境

* CPU: Snapdragon X 12-core X1E80100（ARM64）
* RAM: 16 GiB
* OS: Windows 11
* LLM実行環境: LM Studio
* モデル: gemma-4-e4b-it
* MCPサーバ: web-search-mcp

## 手順

### 1. LM Studio のインストール

まずはLLM実行環境として LM Studio をインストールします。

（注意） ARM64環境では、 https://lmstudio.ai/ の上部にある「Download」から **ARM64 (Snapdragon)** を選択する必要があります。

![](screenshot4-install-lmstudio-arm64.png)

インストール後は（必須ではないですが）設定画面から日本語表示にもできます。

![](screenshot3-lmstudio-lang-ja.png)

**モデルのダウンロード**

LM Studioの左側にあるロボットの顔のようなアイコンをクリックして、 `lmstudio-community/gemma-4-E4B-it-GGUF` をインストールします。

![](screenshot5-download-gemma4.png)

### 2. Node.js / Git のインストール

web-search-mcp のインストールに必要な Node.js と Git をインストールします。

**Node.js**

MCPサーバはNode.jsで動作するため、事前にインストールします。

```bash
winget install OpenJS.NodeJS.LTS
```

確認：

```bash
node -v
npm -v
```

2026/04/09時点では、 node は `v24.14.1` 、 npm は `11.11.0` でした。

**Git**

Gitも（インストールしていなければ）次でインストールします。

```bash
winget install Git.Git
```

### 3. web-search-mcp のインストール

Web検索用のMCPサーバである web-search-mcp をインストールします。

[https://github.com/mrkrsl/web-search-mcp](https://github.com/mrkrsl/web-search-mcp)

設置用のディレクトリを準備して（図では `C:\Users\kamiy\opt\` ）、MCPサーバのリポジトリにクローンとビルドまでやります。

（注意） 公式の手順ではリリースページからzipファイルをダウンロードする手順になっています。が、自分の環境では **git clone する必要** がありました。 zip ファイルからだと最後の build で止まってしまいました。

```bash
git clone https://github.com/mrkrsl/web-search-mcp.git
cd web-search-mcp
npm install
npx playwright install
npm run build
```

![](screenshot2-install-web-search-mcp.png)

### 4. LM Studio に web-search-mcp を登録する

LM Studioのチャットで、右側の金槌アイコンのタブの「Install」→「MCP, Edit mcp.json」を選択して、次のような内容にします。

```
{
  "mcpServers": {
    "web-search": {
      "command": "node",
      "args": ["MCPサーバを導入したディレクトリ\\dist\\index.js"]
    }
  }
}
```

図の例では `"C:\\Users\\kamiy\\opt\\web-search-mcp\\dist\\index.js"` となります。

あと、スライダーをクリックして、有効化するのもお忘れなく。

![](screenshot6-edit-mcpjson.png)

### 5. Web検索つきチャットの実行例

この状態でLM Studio でチャットすると、必要に応じて web-search-mcp を利用するようになります。

![](screenshot1-chat-w-websearch.png)

## まとめ

* ARM64 Windows（Snapdragon）、 RAM 16GiB でも Gemma 4 E4B は快適に動作する。
* しかも、Web 検索付きでチャットできる

### おわりに

Gemma 4 E4B をウェブ検索付き実行すると、ほんとに一昔前の商用サービスみたいなものが、ノートPCで動いてるんだなと感動しました。

軽量モデル単体では難しいタスクでも、外部ツール（Web検索）と組み合わせることで実用的な応答が可能になるの圧巻です（ちょっと驚き屋が入ってるかも）。

今回の構成は、その変化を実感できる良い例でした。
